""" Debugging

La acción de debugger un código es la de solucionar los errores que se
encuentran en este código, esta es una habilidad realmente sencilla de
aprender, y su ejecución no es otra cosa que el estructurar la búsqueda de
errores de manera que se puedan encontrar de una manera ordenada y entendible,
la experiencia y la herramientas de debugg nos pueden ayudar a solucionar
problemas de código para obtener los resultados deseado de una manera más fácil
y entendible.

""" Métodos útiles para hacer debugging

console.log() : Muestra un mensaje por consola, esta función es útil para
mostrar variables de un código y comprobar el valor de dichas variables, poner
un console.log() de manera estratégica puede resolver la mayoría de los
problemas en un bloque de código, ya que nos permite hacer conocer el valor de
las variables en tiempo de ejecución.

console.clear() : Limpia el historial de la consola.

console.log(typeof [variable]) : typeof es un método nativo de JavaScript para
poder identificar el tipado de una variable, recordemos que en JS hay 6 tipos
de datos primitivos: Boolean, Null, Undefined, Number, String, Symbol,
y Object.
